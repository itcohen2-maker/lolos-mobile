# Salinda Brand Assets

This folder contains the first branding pack for `Salinda / סלינדה`.

## Brand Guide

- [`BRAND-MINI-GUIDE.md`](BRAND-MINI-GUIDE.md) - Selected branding direction based on Option 06.

## View everything in the browser (recommended)

Double-click [`preview.html`](preview.html) or drag it into Chrome/Edge/Firefox.  
All images use **relative** paths (`src="salinda-logo-final.svg"` etc.), so they work as long as the HTML file stays in this folder.

## Concepts

- [`salinda-logo-concept-1.svg`](salinda-logo-concept-1.svg) - Card + sparkle with motion trail, bold playful wordmark.
- [`salinda-logo-concept-2.svg`](salinda-logo-concept-2.svg) - Friendly card badge with confetti style accents (recommended base).
- [`salinda-logo-concept-3.svg`](salinda-logo-concept-3.svg) - Dynamic swoosh mark with energetic motion.

## Final Selection

- [`salinda-logo-final.svg`](salinda-logo-final.svg) - Finalized version based on concept 2 for strong readability and playful tone.
- [`salinda-app-icon-variant.svg`](salinda-app-icon-variant.svg) - Icon-only variation suitable for app icon exploration.

## Quick Usage Rules

- Dark background: use `salinda-logo-final.svg` as-is.
- Light background: add a dark outer stroke or place inside a dark rounded container.
- Minimum widths:
  - Full logo: `220px`
  - Icon variant: `64px`
