# Salinda Brand Mini Guide (Option 06)

This guide defines the visual identity for `Salinda / סלינדה` based on `salinda-option-06.svg`.

## 1) Brand Positioning

- Personality: smart, playful, confident, family-friendly.
- Promise: quick fun with thinking and math.
- Voice: clear, upbeat, encouraging, not childish.

## 2) Core Logo

- Master logo file: `salinda-option-06.svg`
- Brand name lockup: `SALINDA` + `סלינדה`
- Primary use: dark backgrounds, game splash, start screen title area.

## 3) Color System

Use these as the default palette:

- Primary Gold: `#F59E0B`
- Background Dark: `#111827`
- Surface Dark: `#1F2937`
- Text Light: `#E5E7EB`
- Optional Accent Cyan: `#22D3EE` (small accents only)

Recommended ratio:

- Dark base/surfaces: 70-80%
- Gold highlights: 15-20%
- Cyan accents: 5-10%

## 4) Typography

- Display/Logo: bold, geometric, high-contrast style (as in Option 06).
- UI Headings: bold and compact.
- Body text: simple and readable, medium weight.
- Keep Hebrew and English visually balanced in size and weight.

## 5) Logo Usage Rules

Do:

- Keep original aspect ratio.
- Use the master logo on dark backgrounds.
- Preserve clear space around logo (at least the height of the Hebrew line).

Do not:

- Stretch or skew the logo.
- Add heavy glow/shadow effects.
- Replace gold with random colors.
- Place logo on noisy backgrounds without a dark container.

## 6) Minimum Sizes

- Full lockup (Salinda + סלינדה): minimum width `220px`
- Icon or compact mark: minimum width `64px`

## 7) Product Mapping

Suggested mapping for current app:

- App background: `#111827`
- Cards/panels: `#1F2937` to `#374151`
- Main CTA / key highlight: `#F59E0B`
- Secondary accent (small): `#22D3EE`
- Primary text: `#E5E7EB`

## 8) Tagline Options

Choose one:

1. `Salinda — משחק חשיבה מנצח`
2. `Salinda — משחקים, מחשבים, מנצחים`
3. `Salinda — קלפים עם שכל`

Recommended default: `Salinda — משחקים, מחשבים, מנצחים`
